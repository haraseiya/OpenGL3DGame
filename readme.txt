【操作方法】
　※PAD操作
　左スティック　→　移動
　右スティック　→　視点移動
　Rボタン	→　弾発射
　Yボタン	→　スペシャルショット（実装途中）
　Startボタン	→　当たり判定用デバッグ画面
　Aボタン	→　シーン遷移

【全体的な流れ】
　目標：ボス敵の討伐
　流れ：ウェーブシステムで大量の敵が迫ってくる、弾幕を回避し銃で敵を殲滅する。

【実装した機能】
　・弾幕処理負荷軽減用のオブジェクトのインスタンス化
　・シャドウ、ブルーム、ポストエフェクトといったシェーダー機能
　・コンポーネントパターンを採用したシステム設計
　・ステートパターンを採用したアニメーション遷移
